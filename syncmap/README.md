# 介紹
使用`syncmap`套件包製作`sync.Map`

# 下載
<!-- > go get -u github.com/a8m/syncmap@master -->
> go install github.com/a8m/syncmap@master

# 生成 sync.Map 指令
> syncmap -name IntMap "map[int]int"
